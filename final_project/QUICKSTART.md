# To Done - Quick Reference Guide

## Keyboard Shortcuts

### Main Menu (Focus Area Selection)
- **1-10** - Select a focus area by number
- **A** - Add (Create) a new focus area
- **S** - Edit (reSign) a focus area name
- **D** - Delete a focus area
- **F** - Help / How to use
- **G** - Go (Exit and save)

### Task Management Menu
- **A** - Add a new task
- **S** - Complete (Strike through) a task
- **D** - Delete a task  
- **F** - Go back to Focus areas

### Quick Actions (Power User Feature)
- **S1** or **S 1** - Complete task #1
- **S2** or **S 2** - Complete task #2
- **D1** or **D 1** - Delete task #1
- **D3** or **D 3** - Delete task #3
- etc.

## Tips for Maximum Productivity

1. **Keep it Simple**: Limit yourself to 2-7 focus areas
2. **Be Specific**: Use clear, actionable task descriptions
3. **Use Quick Actions**: Master S1, D2 shortcuts to work faster
4. **Complete Regularly**: Mark tasks done frequently to see progress
5. **Clean Slate**: Completed tasks auto-delete on exit for fresh starts

## Focus Area Limits

- **Maximum**: 10 focus areas
- **Warning at**: 8 focus areas (80% capacity)
- **Why**: Research shows optimal focus with 7±2 categories

## Data Management

- **Auto-save**: Tasks save automatically when you exit
- **Auto-cleanup**: Completed tasks removed on exit
- **Storage**: Human-readable JSON in `tasks.json`
- **Backup tip**: Copy `tasks.json` to backup your data

## Keyboard Layout (ASDF Home Row)

The keys are positioned for easy access from standard keyboard position:
```
     A S D F
     │ │ │ └─ Back/Help
     │ │ └─── Delete
     │ └───── Complete/Edit
     └─────── Add/Create
```

Your pinky to index fingers rest naturally on these keys!

## Common Workflows

### Adding Multiple Tasks Quickly
1. Enter a focus area
2. Press **A**, type task, press Enter
3. Press **A**, type task, press Enter
4. Repeat!

### Completing Multiple Tasks Fast
1. Enter a focus area (see numbered task list)
2. Type **S1** to complete first task
3. Type **S2** to complete second task
4. Continue!

### Daily Review
1. Launch To Done
2. Check each focus area for task counts
3. Enter areas with active tasks
4. Complete or delete as needed
5. Exit (completed tasks auto-cleanup)

## Safety Features

- **Confirmation prompts** for all deletions
- **Task count warnings** before deleting focus areas
- **Graceful shutdown** on Ctrl+C (saves everything)
- **Limit warnings** when approaching 10 focus areas
